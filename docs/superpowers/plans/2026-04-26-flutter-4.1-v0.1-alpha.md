# Flutter 4.1 v0.1-alpha Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 `apps/mobile` 初始化为可运行 Flutter 工程，具备三 Tab 基础导航（喵测/问题库/我的），并在启动时完成匿名登录（`/v1/auth/anonymous`）与 token/user_id 本地缓存，以便后续 4.2/4.3 直接接入录音/上传/推理。

**Architecture:** 使用 Vanilla Flutter（MaterialApp + Scaffold + BottomNavigationBar + IndexedStack）；新增轻量 `ApiClient`（http + JSON）与 `LocalStore`（shared_preferences）封装；App 启动时在 `HomeScreen` 触发 `ensureAnonymousAuth()`（无 token 时拉取并落盘）；`MeScreen` 展示 user_id 并提供清缓存按钮。

**Tech Stack:** Flutter (Dart) + package:http + shared_preferences。

**Files:**
- Modify/Create: `apps/mobile/lib/api/api_client.dart`

- [ ] **Step 1: 实现 ApiClient 与 baseUrl**

  ApiClient({required this.baseUrl, required this.store});

- [ ] **Step 2: ensureAnonymousAuth()**

逻辑：
- 若 store 里已有 token → return
- 否则 POST `/v1/auth/anonymous`
