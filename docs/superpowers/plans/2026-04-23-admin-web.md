# 喵懂 Admin Web Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 交付一个“奶油系卡哇伊风”的运营后台 Admin Web（静态单页），支持口令登录、Dashboard/Problems/Releases/Metrics，并由 Go API 托管访问。

**Architecture:** 静态单页（`apps/admin`）通过 `fetch` 调用同域的 Admin API（`/admin/*`），使用 `X-Admin-Token` 口令鉴权。Go API 额外提供静态文件服务入口（建议路径：`/admin/ui/`）以避免与现有 `/admin/problems` 等 API 冲突。

**Tech Stack:** HTML/CSS/Vanilla JS（无构建链） + Go net/http 静态托管 + 既有 Postgres/Admin API。

- [ ] **Step 3: 实现 adminui.Handler()（定位 apps/admin 并 FileServer）**

实现要点：
1) 使用 `runtime.Caller` 从 `static.go` 位置计算仓库根目录（与 `DefaultSeedPath` 思路一致）
2) `apps/admin` 必须存在，否则返回错误（让 CI 明确失败）
3) 返回 `http.StripPrefix("/admin/ui/", http.FileServer(http.Dir(adminDir)))`
4) 额外处理：当访问 `/admin/ui/` 时返回 `index.html`（为了更友好；即使采用 hash 路由也安全）

```go
package adminui

import (
	"fmt"
	"net/http"
	"os"
	"path/filepath"
	"runtime"
)

func Handler() (http.Handler, error) {
	_, thisFile, _, ok := runtime.Caller(0)
	if !ok {
		return nil, fmt.Errorf("runtime.Caller failed")
	}
	// .../services/api/internal/http/adminui/static.go -> repo root
	root := filepath.Clean(filepath.Join(filepath.Dir(thisFile), "..", "..", "..", "..", ".."))
	adminDir := filepath.Join(root, "apps", "admin")
	if _, err := os.Stat(filepath.Join(adminDir, "index.html")); err != nil {
		return nil, fmt.Errorf("admin ui missing: %w", err)
	}

	fs := http.FileServer(http.Dir(adminDir))

	// 让 /admin/ui/ 默认落到 index.html
	return http.StripPrefix("/admin/ui/", http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		if r.URL.Path == "" || r.URL.Path == "/" {
			r.URL.Path = "/index.html"
		}
		fs.ServeHTTP(w, r)
	})), nil
}
```

## Task 3: Admin Web 鉴权与路由（口令登录）

**Files:**
- Create: `apps/admin/admin.js`

- [ ] **Step 1: 写最小前端“登录页”渲染**

要求：
1) 若 `localStorage.miaodong_admin_token` 不存在 → 显示口令输入框
2) 输入后保存并进入 Dashboard
3) 后续所有请求自动带 `X-Admin-Token`

核心代码骨架（实现时可拆成小函数，但不得引入构建链）：

```js
const LS_TOKEN_KEY = "miaodong_admin_token";

function getToken() {
  return localStorage.getItem(LS_TOKEN_KEY) || "";
}
function setToken(t) {
  localStorage.setItem(LS_TOKEN_KEY, t);
}
function clearToken() {
  localStorage.removeItem(LS_TOKEN_KEY);
}

async function apiFetch(path, { method = "GET", body } = {}) {
  const headers = { "Content-Type": "application/json" };
  const token = getToken();
  if (token) headers["X-Admin-Token"] = token;
  const res = await fetch(path, { method, headers, body: body ? JSON.stringify(body) : undefined });
  if (res.status === 401) {
    clearToken();
    renderLogin("口令无效或已过期，请重新输入");
    throw new Error("unauthorized");
  }
  if (!res.ok) throw new Error(await res.text());
  return await res.json();
}
```
