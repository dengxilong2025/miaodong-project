# Flutter 4.5 Share Copy + System Share v0.1-alpha Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 Flutter 结果页实现“复制分享文案 + 调起系统分享”，优先使用 `/v1/templates/result?problem_id=...` 返回的 `share_text_template`，失败时回退到客户端默认模板，并默认附带 `#喵懂 #喵测`。

**Architecture:** 保持 v0.1-alpha 轻量实现，不新增独立分享 service；直接在 `ResultScreen` 内生成分享文案并调用 `Clipboard` / `SharePlus.instance.share()`。有 `problemId` 时先用 `ApiClient.getJson()` 拉模板；无 `problemId` 或模板失败时回退本地模板。

**Tech Stack:** Flutter (Dart) + `share_plus` + `flutter/services.dart` Clipboard。

## Task 4: ResultScreen 增加“系统分享”


- [ ] **Step 2: 实现系统分享**

```dart

  final text = await _buildShareText();
  await SharePlus.instance.share(ShareParams(text: text));
```
